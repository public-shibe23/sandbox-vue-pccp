# sandbox-vue-pccp
Demo for Presentational and Container Component Pattern.

[Vue.js+Vuexでasync/awaitを使ったPresentational and Container Component Pattern(サンプルつき)](https://qiita.com/shibe23/items/52ab65710678031b2ded)

## Project setup
```
npm install
```

### Run demo
```
npm run serve
npm run db
```
or

```
npm run demo
```
